A simple way to add social links (like: facebook, twitter, tumblr, last.fm, flickr, plurk, etc) to your Links (Bookmarks?!). Uma forma ~simples~ de adicionar informações aos links (bookmarks?!), como: twitter, tumblr, last.fm, facebook, flickr, orkut, etc. Ainda testando.


== Description ==
Este plugin é um tutorial que apresenta o desenvolvimento de plugins no wordpress, ele executa 3 operações.
1. Acrescentará ao final de cada comentário o texto: "O conteúdo deste comentário é de responsabilidade de: {AUTOR}"
2. Criará uma tabela nova que irá armazenar uma contagem de visitas a cada post, e mostrará esta contagem logo ao final do POST.
3. Substituir a palavra XXX por um texto especificado
